Syen: E-ink Smart Watch eCommerce Responsive Site

My Process

Objectives / Project Goals

The primary user goal guiding this project was centered around creating an eCommerce shopping experience via responsive web app that let users browse a range of smart watches and purchase them efficiently and easily. Emotionally connecting to users via branding / multimedia / imagery and sense of community was also strongly considered throughout design development.

Primary Research and Insights

Primary research was done doing SWOT analyses on a number of eCommerce focused sites of brands that hold similar values and that overlap with the customer base demographics of SYEN.

User Stories and Flow Map

A set of user stories was developed before doing an initial map out of the site. Selecting the correct success criteria for each user story was important to define as well. There are a lot of factors that can go into a user not completing a purchase or not ultimately connecting with a brand by signing up for their email newsletter so it was important to get these stories as accurate as possible from the beginning.

Early UI Idea Generation

Once a clear framework was established via the initial user stories and flow map I turned my attention to quickly getting some ideas and notes down via some hand sketches done on my iPad in the Procreate app. Along with thinking about user interactions I also started developing the brand identity in parallel.

Brand Development

Development of the SYEN brand was built around its core value of creating a smart watch brand that caters to tech savvy young people who want healthy boundaries with their tech. The focus then was creating branding imagery and an overall feel that is minimalistic, feels future forward and hopeful, and has a universal stylish appeal.

Build Initial Wireframes

User Testing Criteria and Results

User testing with the initial prototype was done with five participants. I made sure to choose people who were already tech savy so as to match my target user market as closely as possible.

Final Designs and Responsive Screens

To make everything as realistic as possible, I also dug back into my industrial design background and built and rendered a set of watches and straps to use as the product content. Building was done in a CAD software called SolidWorks and rendering in Keyshot.

Lessons Learned and Areas for Improvement

As this project would continue to evolve there are a number of things I would continue to test and iterate on. Specifically email newsletter sign-ups and checkout are generally most important to eCommerce, so most of the focus would be there.
